Les PWA (Progressive Web Apps) sont des applications web qui combinent les meilleures caractéristiques des applications mobiles et des sites web classiques. Elles s'installent directement depuis un navigateur web, sans passer par un magasin d'applications, et offrent une expérience utilisateur similaire à celle d'une application native.

Grâce à des technologies web modernes comme les service workers, les PWA peuvent fonctionner hors ligne, envoyer des notifications push et accéder à des fonctionnalités matérielles de l'appareil. Elles sont plus rapides, plus fiables et plus engageantes que les sites web traditionnels, tout en étant plus légères et plus faciles à mettre à jour que les applications natives.

Les PWA sont une solution de développement "multi-plateforme" qui permet de cibler les utilisateurs sur mobile et desktop avec une seule base de code, réduisant ainsi les coûts de développement et de maintenance.



Sur Android (avec le navigateur Chrome) Ouvrez le navigateur Chrome sur votre téléphone. Rendez-vous à l'adresse de votre application sur GitHub Pages : https://.github.io// Appuyez sur le bouton du menu (les trois points verticaux en haut à droite). Cherchez l'option "Installer l'application" ou "Ajouter à l'écran d'accueil". Une fenêtre de confirmation apparaîtra. Appuyez sur "Installer". C'est fait ! L'icône de l'application "Allez-y" apparaîtra sur votre écran d'accueil, comme n'importe quelle autre application.
Sur iPhone / iOS (avec le navigateur Safari) Ouvrez le navigateur Safari sur votre iPhone (cette fonction n'est pas disponible sur Chrome pour iOS). Rendez-vous à l'adresse de votre application sur GitHub Pages. Appuyez sur le bouton Partager (l'icône est un carré avec une flèche qui sort vers le haut, située en bas au centre de l'écran). Faites défiler la liste des options et appuyez sur "Sur l'écran d'accueil". Vous pouvez confirmer ou modifier le nom de l'icône, puis appuyez sur "Ajouter" en haut à droite. Et voilà ! L'icône de l'application apparaîtra sur votre écran d'accueil.
